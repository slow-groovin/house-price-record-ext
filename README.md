# ![](./public/icon/24.png)真实房价助手

这是一款浏览器插件(Chrome 内核, Chrome, Edge, ...), 帮助正在关注二手房市场的购房者/租房者:

1. 信息收集: 自动化一键收集记录(链家平台)关注范围内的小区/房源状态, 挂牌价
2. 数据记录: 记录入浏览器 IndexedDB 数据库
3. 数据管理: 查看, 筛选, 可视化数据

所有的运行过程和数据都在浏览器内
